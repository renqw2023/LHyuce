# 快速开始指南 - 5分钟上手

## 启动系统

```bash
streamlit run dashboard.py --server.headless true
```

浏览器会自动打开UI界面。

## 最简单的使用方式（推荐）⭐

### 步骤1：打开UI
运行上面的命令，等待浏览器打开

### 步骤2：点击"执行中心"
在左侧导航栏选择 **"执行中心"**

### 步骤3：运行V7预测

## 三种使用方式

### 方式1：只用V7（最快）⭐
```
执行中心 → 运行V7预测 → 完成
```
**适合：**
- 只关心特码预测
- 需要快速结果
- 想要高准确率

### 方式2：只用V6（传统）
```
执行中心 → 优化策略 → 运行每日分析 → 查看各页面
```
**适合：**
- 需要完整复盘
- 想要多维度分析
- 习惯原有流程

### 方式3：V6+V7（最全面）
```
1. 先运行V7预测
2. 再运行每日分析
3. 对比两个预测
```
**适合：**
- 想要最全面的分析
- 对比不同算法
- 深度研究用户

## 常见问题

**Q: 我还需要运行"每日分析"吗？**
A: 不一定。如果只要特码预测，用V7就够了。

**Q: V7和V6哪个准？**
A: V7准确率76%，V6准确率41%。推荐用V7。

**Q: 可以同时用吗？**
A: 可以！两个系统独立运行，互不影响。

**Q: 多久运行一次？**
A: 建议每期开奖后运行一次V7预测。

**Q: 需要优化策略吗？**
A: V7已经优化好了，可以直接用。如果想重新优化，点击"运行V7特码优化"。
